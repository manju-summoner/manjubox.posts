## 追加
- 「ウェーブクリッピング」「グラデーションマップ」「ゴッドレイ」「熱揺らぎ」「条件分岐ダイナミクス」エフェクトを実装
- 音声読み込み「MIDI」を実装
  - 上記2項目はroutersysさんによる実装です
   - YMM4では、YMM4コミュニティプラグインを通じて有志の方々による機能追加が行われています。
   - YMM4コミュニティプラグインのGitHubリポジトリは[こちら](https://github.com/manju-summoner/YukkuriMovieMaker.Plugin.Community)
- 「出力を分岐」「出力を切り替え」「出力を合成」エフェクトを追加
- 「Irodori-TTS」「GrokTTS」に対応
- AIテキスト補完で「CodexCLI」「ClaudeCode」に対応
- 文字制御タグ追加
  - ルビタグ&lt;rb&gt;でフォント、スタイル、色を指定できるようにした
  - &lt;ls&gt;タグで文字の間隔を指定できるようにした
  - &lt;@&gt;タグでフォント名・スタイルを指定できるようにした
  - &lt;s&gt;&lt;@&gt;タグの装飾文字列に0-6の数字を含めることで、装飾種別（影/縁取り）を文字単位で切り替えられるようにした
  - &lt;#&gt;色タグで色名（"red", "blue"等）による指定に対応
- 模様のプリセットを保存できるようにした
- FFmpeg動画/音声読み込み機能を実装
- 設定 → プラグイン → 動画 / 音声 / 画像読み込みプラグインから、組み込みプラグインの優先度を変更できるようにした
- gemini-3.1-flash-tts-previewに対応
- タイムラインアイテムの背景をグラデーションと単色のどちらで描画するか切り替えられるようにした（設定 → 全般 → タイムライン → アイテムの背景をグラデーションで描画）
- 「起き上がりながら登場退場」エフェクトに起き上がる方向（奥から／手前から）を切り替えられるようにした
- シーンごとに画面の背景色を指定できるようにした（動画の設定 → 背景色）
- 選択中のアイテムの長さを元のコンテンツ長にリセットするコマンド「アイテムの長さをリセット」を追加
- UIの拡大率を100%にリセットするコマンド「UIの拡大率を100%にリセット」を追加（レイアウトメニュー）
- 「ノイズ歪み」「ノイズ色ずれ」エフェクトに「ぼかし度」を追加
- 「縁取り（内側）」を軽量化
- 各種%系項目のスライダー上下限を-100～100または0～100に統一
- グラデーションエディタで中クリックでポイントを削除出来るようにした
- COEIROINKv2のPronounceEditor上でマウスホイールスクロールした際、横スクロールするようにした
- VOICEVOX/COEIROINK設定画面に、設定方法のヘルプページへのリンクを追加
- [API] 後続のエフェクトに値を渡せるようにした。
  - DrawDescription: Set/Get/TryGetCustomValue
  - EffectDescription: Get/TryGetCustomValue
- [API] FileSelectorコントロールに、カスタムサムネイルローダーを外部から指定できるようにした。
- [API] FileSelectorコントロールのドロップダウン内アイテム右端に、外部から任意の追加コンテンツ（バッジ等）を表示できるようにした。
  - FileSelector.TrailingContentTemplate: DataTemplate 型 DependencyProperty。DataContextはFileItem（FilePathから拡張子判定等が可能）
- [API] FileSelectorコントロールで使用するファイルグループ（お気に入り・最近使ったファイル）を、外部から任意に定義できるようにした。
  - 事前にFileGroups.RegisterCustomGroup(...);を呼んでグループを登録する必要があります。
- [API] グラデーションエディタをプラグインから利用可能にした。
- [API] CreateItemAudioSourceで、エフェクト適用前のコンテンツ長に対応するstreamを取得できるようにした。
- [API] JobObjectをPlugin.dllへ移動してpublic化した。
## 修正
- 発音欄の再生成ボタンをクリックしても音声が再生成されないことがある問題を修正
- 模様「シーン」の再生速度が100倍になっている問題を修正
- シーンアイテムの「波形」欄のテキストが誤って設定されている問題を修正
- セリフに制御タグを入力した際、タグが読み上げ対象になってしまう問題を修正
- &lt;rb&gt;タグの親文字・ルビ部分に、テキストの部分装飾が出来ない問題を修正（ルビの打ち消し・下線は非対応）
- &lt;rb&gt;タグのルビ色がパースできない問題を修正
- タイムライン上のアイテム同士に隙間が出来ることがある問題を修正
- テキストの描画エリアが拡張されている問題を修正
- あいうえお口パク情報が生成されないことがある問題を修正
- 存在しないプラグインの図形アイテムを含むプロジェクトファイルを開けない問題を修正
- ツールウィンドウが増殖する問題の対策をした
