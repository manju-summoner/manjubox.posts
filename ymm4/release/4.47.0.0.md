## 追加
- VoiSonaTalkに対応
- Kotodamaに対応
- テンプレートツールのテンプレート一覧をツリー化できるようにした
- テンプレートツールに検索機能を追加
- テンプレートの追加レイヤーを指定できるようにした
- テンプレートから複数のエフェクトを追加した際、一番下のエフェクトが選択されるようにした
- プレビューの再生速度変更時、音程が維持されるようにした
  - 設定 → 全般 → プレビュー → 再生速度変更方式から設定を変更可能です
- CtrlやShiftキーを押しながらフローティングウィンドウドラッグすることで、ドッキングを無効化しながらウィンドウを移動出来るようにした
- 音声の逆再生に対応
- フォントをグループ化できるようにした（設定 → フォント → グループ）
- AIテキスト補完ツールで主要なLLMサービスに対応
- .NET10へ移行
## 修正
- 音量解析ツールの結果が正常に描画されないことがある問題を修正
- シーン名・レイアウト名の変更時、ウィンドウを閉じて操作をキャンセルできない問題を修正
- 音声エフェクト一覧でにプラグイン由来のエフェクトであることを示すマークが表示されない問題を修正
- IME変換中にコマンドが実行される問題を修正
- 「色覚異常」エフェクトの名前を「色覚特性シミュレーション」に変更
- Koemotionのサービス終了に伴い、関連機能を削除
- UIの拡大率が100%未満に設定されている場合、一部の線が描画されない事がある問題を修正
- アイテムを右クリックした際のコンテキストメニューが表示されるタイミングを調整
- アイテムがドラッグ状態のままになることがある問題を修正
- 台本編集ウィンドウが開かれていない状態で台本を開くコマンドを実行した際、台本選択をキャンセルしてもウィンドウが非表示にならない問題を修正
- [API] YukkuriMovieMaker.Controls.dll内に定義されているUIをYukkuriMovieMaker.Plugin.dllに移動
  - 名前空間に変更はありません。
  - 多くの場合特別な対応は不要ですが、一部のプラグインで再ビルドが必要になる可能性があります。
