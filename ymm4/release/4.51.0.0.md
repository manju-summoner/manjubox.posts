## 追加
- テキストの装飾に「打ち消し線」「下線」を追加
- テキストアイテム・ボイスアイテムで使用可能な、文字制御タグを実装
  - AviUtlやPSDToolKitの制御文字の仕様を参考にしつつ、独自に拡張した物になっています。
  - タグ一例
    - &lt;#ffffffcc,ff0000ff&gt; 色変更（文字色、装飾色 / 16進数RGB, RGBA）
	- &lt;s100, Arial, BISU&gt; サイズ・フォント・スタイル（太字, 斜体, 打ち消し線, 下線）変更
	- &lt;p+10, -5&gt; 描画座標変更
	- &lt;pp+100, +0&gt; 描画座標変更（フォントサイズ基準（％））
	- &lt;rot45&gt; 回転（°）
	- &lt;rb東京, とうきょう&gt; ルビ
	- &lt;r1&gt; 表示速度（文字/秒）
	- &lt;w1&gt; 表示待機（秒）
	- &lt;c1&gt; 表示クリア（秒）
  - その他仕様
    - &lt;s*=1.2&gt; のように、+= -= *= /= の演算子を使用して現在の値に対する相対的な数値変更が可能です。（&lt;w&gt; &lt;c&gt;タグを除く）
	- &lt;s&gt; &lt;#&gt; のように、パラメータを省略した場合、デフォルトの値にリセットされます。
	- &lt;s 12, Arial&gt; &lt;rot 45&gt; のように、パラメータの前後にスペースを入れることが可能です。
	- \&lt;s&gt; のように、タグの前に \ を置くことでタグをそのまま表示することが可能です。
	- &lt;rb 1\,000円,せんえん&gt; のように、カンマの前に \ を置くことでルビタグ内でカンマを使用可能です。
	- 詳細な仕様は[文字制御タグでテキストを装飾/表示タイミングをコントロールする](https://manjubox.net/ymm4/faq/editing/text_control_tags)を参照してください。
- テキストアイテムのテキスト欄で、複数のアイテムを同時編集できるようにした
## 修正
- テキストを文字毎に個別に表示し、表示/非表示アニメーションを設定した際、フェードが正常に動作しない問題を修正
- 動く立ち絵素材編集ボタンのアイコンが表示されない問題を修正
- 中心位置エフェクトで例外が発生することがある問題を修正
- Hyperlinkでキー操作を行った場合、例外が発生することがある問題を修正
