# 文字制御タグ リファレンス

テキスト中に制御タグを埋め込むことで、文字単位で色・フォント・位置・回転・表示タイミングなどを変更できます。

---

## 共通ルール

- タグは **大文字・小文字を区別しません**（`<s>` と `<S>` は同じ）
- 不正な構文のタグはそのまま文字として表示されます
- タグをエスケープ（文字として表示）するには先頭に `\` を付けます: `\<s20>` → `<s20>` と表示
- パラメータの前後に **スペース** を入れても動作します（例: `<s 120>`, `<rb 東京, とうきょう, 30>`）
- 多くのタグで **四則演算** による相対指定が可能です:
  - `+=N` 加算
  - `-=N` 減算
  - `*=N` 乗算
  - `/=N` 除算
  - `<w>` および `<c>` タグは四則演算非対応

## 色タグ `<#...>`

文字色・装飾色（縁取り・影）を変更します。

### 書式

| 書式 | 説明 |
|------|------|
| `<#rrggbb>` | 文字色を16進数で指定（例: `<#ff0000>` = 赤） |
| `<#rrggbbaa>` | 文字色+不透明度（`aa`: `00`=透明, `ff`=不透明） |
| `<#rrggbb,rrggbb>` | 文字色, 装飾色をカンマ区切りで指定 |
| `<#色名>` | 色名で指定（例: `<#red>`, `<#DarkOrange>`） |
| `<#>` | 文字色・装飾色をリセット |

### 短縮形式

| 文字数 | 展開ルール | 例 |
|--------|-----------|-----|
| 1文字 `x` | `xxxxxx` | `<#f>` → `<#ffffff>` |
| 2文字 `xy` | `xyxyxy` | `<#a0>` → `<#a0a0a0>` |
| 3文字 `rgb` | `rrggbb` | `<#f00>` → `<#ff0000>` |
| 4文字 `rgba` | `rrggbbaa` | `<#f008>` → `<#ff000088>` |

### 色名による指定

CSSの色名が利用できます。大文字・小文字は区別しません。

- 例: `<#red>`, `<#blue>`, `<#DarkOrange>`, `<#LightGray>`
- 色名と装飾色名の組み合わせも可能: `<#red,navy>` = 文字色赤、装飾色ネイビー

### 例

```
<#ff0000>赤いテキスト<#>ここから通常色
<#00ff00,0000ff>緑の文字に青い装飾<#>
<#f00>短縮形式で赤<#>
<#red>色名で赤<#>
<#DarkOrange,Black>濃いオレンジに黒装飾<#>
```

## フォントタグ `<s...>`

フォントサイズ・フォント名・太字/斜体を変更します。

### 書式

| 書式 | 説明 |
|------|------|
| `<sN>` | フォントサイズをNに設定 |
| `<sN,フォント名>` | サイズとフォントを変更 |
| `<sN,フォント名,BISU>` | サイズ・フォント・装飾を変更 |
| `<s,フォント名>` | フォント名のみ変更 |
| `<s,,B>` | 太字のみ変更 |
| `<s>` | すべてリセット |

### 装飾の指定

3番目のパラメータに以下の文字を含めます:

- `B` — 太字
- `I` — 斜体
- `S` — 打ち消し線
- `U` — 下線
- 組み合わせ可能: `BI`, `BS`, `BU`, `BISU` など
- 装飾種別（影/縁取り）を 0-6 の数字で指定可能:
  - `0` なし、`1` 影、`2` 影（薄）、`3` 丸縁取り、`4` 丸縁取り（薄）、`5` 角縁取り、`6` 角縁取り（薄）
  - 例: `<s60,メイリオ,B3>` = 60pt メイリオ 太字 + 丸縁取り
  - 数字を指定しなければアイテム既定の装飾種別が使われます

### 四則演算

フォントサイズに四則演算が使えます:

```
<s*=2>2倍サイズ<s>
<s+=10>10ポイント加算<s>
```

### 例

```
<s80>大きな文字<s>通常サイズ
<s,メイリオ>フォント変更<s>
<s60,HGPゴシックE,BI>太字斜体の60pt<s>
<s,,S>打ち消し線<s>
<s,,U>下線<s>
<s,,BIS>太字+斜体+打ち消し線<s>
<s,,BISU>太字+斜体+打ち消し線+下線<s>
```

## フォント名・スタイルタグ `<@...>`

フォントサイズを変えずにフォント名とスタイル（太字/斜体/打ち消し線/下線）だけを変更します。`<s>` でも同じことはできますが、サイズ部分のカンマを省略できる分こちらが簡潔です。

### 書式

| 書式 | 説明 |
|------|------|
| `<@フォント名>` | フォント名のみ変更 |
| `<@フォント名,BISU>` | フォント名と装飾を変更 |
| `<@,BISU>` | 装飾のみ変更（フォント名は現在の状態を維持） |
| `<@+B>` / `<@-B>` | 装飾を個別にオン/オフ（カンマ省略可） |
| `<@,+B-I>` | 装飾を個別にオン/オフ（カンマ区切り版） |
| `<@>` | フォント名・装飾をリセット（フォントサイズには影響しない） |

### 装飾の指定

装飾は2つの書き方をサポートします。

#### 全置換モード（`<@,BISU>`）

カンマの後に装飾文字を並べます。**書いた装飾はオン、書かなかった装飾は強制オフ**になります。

- `B` — 太字
- `I` — 斜体
- `S` — 打ち消し線
- `U` — 下線
- 組み合わせ可能: `BI`, `BS`, `BU`, `BISU` など
- 例: `<@,B>` は太字のみで、斜体・打ち消し線・下線はオフ
- 装飾種別（影/縁取り）を 0-6 の数字で指定可能:
  - `0` なし、`1` 影、`2` 影（薄）、`3` 丸縁取り、`4` 丸縁取り（薄）、`5` 角縁取り、`6` 角縁取り（薄）
  - 例: `<@メイリオ,B3>` = メイリオ + 太字 + 丸縁取り
  - 数字を指定しなければアイテム既定の装飾種別が使われます

#### 差分モード（`<@+B>`, `<@-I>`）

`+` で装飾をオン、`-` でオフにします。**指定しなかった装飾は現状維持**です。

- `<@+B>` — 太字をオンにする（他は維持）
- `<@-B>` — 太字をオフにする（他は維持）
- `<@+B+I>` または `<@+BI>` — 太字と斜体を両方オン（連続する装飾文字はまとめて指定可能）
- `<@+B-I>` — 太字をオン、斜体をオフ
- `<@,+B>` — フォント名は維持したまま太字をオン（カンマ区切り版でも同じ）
- `<@+3>` — 装飾種別を 3（丸縁取り）に設定（他は維持）
- `<@-3>` — 装飾種別をリセット（アイテム既定値に戻す）

第1引数が `+` または `-` で始まっていれば差分モードと判定されるため、`<@+B>` のようにカンマを省略できます。

他の制御タグの四則演算と表記を揃えるため、`+=` / `-=` 形式も同じ意味で使えます。

- `<@+=B>` は `<@+B>` と同義（太字オン）
- `<@-=B>` は `<@-B>` と同義（太字オフ）
- `<@+=BI-=SU>` は `<@+BI-SU>` と同義

### 注意事項

- `<@>` は **フォントサイズをリセットしない** ため、`<s60>` で大きくした状態のままフォント名だけ切り替えたいときに便利です。
- フォントサイズも一緒に変えたい場合は従来通り `<s>` タグを使ってください。

### 例

```
<@メイリオ>メイリオで表示<@>標準フォントに戻る
<@HGPゴシックE,BI>太字斜体のHGPゴシックE<@>
<@,U>下線のみ追加<@>
<s60><@メイリオ>大きいメイリオ<@>大きい標準フォント<s>
通常<@+B>太字<@+I>太字+斜体<@-B>斜体のみ<@>通常
<@メイリオ,+B>メイリオ太字（既存装飾は維持）<@>
<@メイリオ,B3>メイリオ+太字+丸縁取り<@>
通常<@,1>影<@,3>丸縁取り<@,5>角縁取り<@>通常
```

## 座標タグ `<p...>`

文字の表示位置をオフセットします。Y軸は下方向が正です（AviUtl互換）。

### 書式

| 書式 | 説明 |
|------|------|
| `<pX,Y>` | 絶対座標で位置を指定 |
| `<p+X,+Y>` | 現在位置からの相対移動 |
| `<p-X,-Y>` | 現在位置からの相対移動（負方向） |

### 四則演算

```
<p*=2,*=2>  現在のオフセットを2倍
<p+=100,+=0>  X方向に100加算
```

### 注意事項

- X座標のオフセットは **改行でリセット** されます
- 符号なし（`<p100,50>`）は絶対座標、符号あり（`<p+100,+50>`）は相対座標として扱われます

### 例

```
通常位置<p+100,+0>右に100移動<p+0,+50>さらに下に50移動
<p200,100>絶対座標(200,100)に配置
```

## フォントサイズ比例座標タグ `<pp...>`

`<p>` と同様に位置をオフセットしますが、値をフォントサイズの **1/100 単位** で指定します。フォントサイズが変わっても相対的な位置関係を保てます。

### 書式

| 書式 | 説明 |
|------|------|
| `<ppX,Y>` | フォントサイズ比例で絶対座標指定 |
| `<pp+X,+Y>` | フォントサイズ比例で相対移動 |

値 `100` = フォントサイズ 1 文字分です。

### 換算例（フォントサイズ 40pt の場合）

| タグ | 実際のオフセット |
|------|----------------|
| `<pp100,0>` | X方向に 40px（= 40 × 100/100） |
| `<pp50,0>` | X方向に 20px（= 40 × 50/100） |
| `<pp+200,+100>` | X方向に 80px、Y方向に 40px |

### 四則演算

`+=` `-=` はフォントサイズ比例で加減算されます。`*=` `/=` は既存のピクセル値に対する倍率として扱います。

```
<pp+=50,+=0>    フォントサイズの半分だけX加算
<pp*=2,*=2>     現在のオフセットを2倍（ピクセル単位の倍率）
```

### 注意事項

- `<s>` タグでフォントサイズを変更した後に `<pp>` を使うと、変更後のサイズが基準になります
- `<p>` と `<pp>` は同じオフセット値を共有します（`<pp>` の後に `<p>` で上書き可能）
- Y軸は `<p>` と同じく下方向が正です（AviUtl互換）

### 例

```
通常<pp+100,+0>1文字分右に移動
<s80>大きい文字<pp+50,+0>半文字分ずれる<s>
```

## 回転タグ `<rot...>`

テキストを回転させます。回転の中心はタグ範囲内のテキストの中心です。

### 書式

| 書式 | 説明 |
|------|------|
| `<rotN>` | N度回転 |
| `<rot>` | 回転をリセット（0度に戻す） |

### 四則演算

```
<rot+=15>A<rot+=15>B<rot>C
```
→ Aは15°、Bは30°（累積）、Cはリセット

### 例

```
<rot30>斜めテキスト<rot>通常テキスト
<rot45>AB<rot>CD        → ABのみ45°回転
<rot180>上下反転<rot>
```

## ルビタグ `<rb...>`

テキストにルビ（振り仮名）を振ります。親文字の上（横書き）または右（縦書き）にルビテキストが表示されます。

### 書式

| 書式 | 説明 |
|------|------|
| `<rb親文字,ルビ>` | 親文字にルビを振る（フォントサイズはデフォルト50%） |
| `<rb親文字,ルビ,N>` | フォントサイズをN%で指定 |
| `<rb親文字,ルビ,N,X,Y>` | フォントサイズN% + ルビ位置をX,Yピクセルずらす |
| `<rb親文字,ルビ,,X,Y>` | デフォルトサイズ + ルビ位置をX,Yピクセルずらす |
| `<rb親文字,ルビ,N,X,Y,フォント,BI,文字色,装飾色>` | ルビ専用のフォント・太字/斜体・文字色・装飾色を指定 |

- `rb` の直後にカンマなしで親文字を記述します
- 親文字はクリーンテキストに挿入され、通常の文字として表示されます
- ルビのフォントサイズは親文字の実際のフォントサイズに対する百分率です
- X,Y はルビの描画位置をピクセル単位でオフセットします（省略可）
- フォント以降の追加パラメータはすべて省略可能です
- `BI` には `<s>` タグと同様に `B` / `I` / `BI` を指定できます（`U` `S` は非対応）
- 文字色・装飾色は `<#...>` タグと同じ16進数形式で指定できます
- 親文字・ルビ文字中にカンマを含めたい場合は `\,` でエスケープします（例: `<rb1\,000円,せんえん>`）

### 四則演算

フォントサイズにはデフォルト値（50%）に対する四則演算が使用可能です。

```
<rb東京,とうきょう,*=2>  → 50 × 2 = 100%
<rb東京,とうきょう,+=20> → 50 + 20 = 70%
<rb東京,とうきょう,-=10> → 50 - 10 = 40%
<rb東京,とうきょう,/=2>  → 50 / 2 = 25%
```

### 例

```
<rb東京,とうきょう>に行く        → 「東京」の上に「とうきょう」（50%サイズ）
<rb漢字,かんじ,30>テスト         → 「漢字」の上に「かんじ」（30%サイズ）
<rb東京,とうきょう,,0,-5>        → デフォルトサイズ、5ピクセル上にずらす
<rb1\,000円,せんえん>            → 「1,000円」の上に「せんえん」（カンマエスケープ）
<s120><rb漢字,かんじ>            → 120%に拡大された「漢字」に対する50%サイズのルビ
<rb東京,とうきょう,50,0,0,游明朝,BI,ff0000,00ffff> → ルビだけ游明朝の太字斜体、文字色は赤、装飾色はシアン
<rb強調,きょうちょう,50,0,0,,B> → ルビだけ太字
<rb斜体,しゃたい,50,0,0,,I,00ff00> → ルビだけ斜体、文字色は緑
```

## 表示速度タグ `<r...>`

文字の表示速度（1文字ずつ表示するアニメーション）を変更します。

### 書式

| 書式 | 説明 |
|------|------|
| `<rN>` | N文字/秒で表示 |
| `<r0>` | 即時表示（残り全文字を一括表示） |
| `<r>` | 表示速度をリセット（アイテムの設定値に戻す） |

### 四則演算

```
<r*=2>2倍速で表示<r>
<r+=5>5文字/秒加算<r>
```

### 例

```
通常速度<r20>高速表示<r>通常に戻る
ゆっくり<r2>表示される<r0>残りは即時
```

## 待機タグ `<w...>`

文字の表示を一時停止します。

### 書式

| 書式 | 説明 |
|------|------|
| `<wN>` | N秒間待機 |
| `<w*N>` | 直前のタイミングタグからの文字数 × N秒 待機 |

### 例

```
こんにちは<w1>（1秒待機）世界
あいう<w*0.5>えお    → 「あいう」の3文字 × 0.5秒 = 1.5秒待機
```

---

## クリアタグ `<c...>`

表示中のテキストを消去し、後続のテキスト表示を開始します。

### 書式

| 書式 | 説明 |
|------|------|
| `<c>` | 即時クリア |
| `<cN>` | N秒かけてクリア |

### 例

```
最初のセリフ<c1>次のセリフ<c>最後のセリフ
```
→ 「最初のセリフ」表示後、1秒で消えて「次のセリフ」が表示。その後即時クリアして「最後のセリフ」が表示。

## 文字間隔タグ `<ls...>`

タグ範囲内の文字間隔（字間、Letter Spacing）をピクセル単位で変更します。アイテム標準の「文字間隔」プロパティ（`LetterSpacing`）の代わりに、文字単位で字間を上書きします。

### 書式

| 書式 | 説明 |
|------|------|
| `<lsN>` | 文字間隔をNピクセルに設定 |
| `<ls>` | 文字間隔をリセット（アイテムの設定値に戻す） |

### 四則演算

`+=` `-=` `*=` `/=` の基準値は **アイテムの「文字間隔」プロパティの現在値** です。タグで一度設定した後はその値が以降の四則演算の基準になります。

```
<ls+=10>10pxだけ字間を広げる<ls>
<ls-=2>2pxだけ字間を詰める<ls>
<ls*=2>字間を2倍に<ls>
<ls/=2>字間を半分に<ls>
```

### 注意事項

- 値はピクセル単位の絶対値です（フォントサイズに連動しません）。
- 範囲内の最後の文字の **trailing spacing**（末尾側余白）として適用されるため、行末や `<ls>` リセット直前の1文字には影響しないことがあります。

### 例

```
通常文字<ls20>字間広め<ls>通常に戻る
<ls0>字間ゼロで詰める<ls>
<ls+=5>標準より5px広い<ls>標準
```

## エスケープ

タグ文字列をそのままテキストとして表示したい場合は、先頭に `\` を付けます。

```
\<#ff0000>  → <#ff0000> と表示される（色は変わらない）
\<s80>      → <s80> と表示される（サイズは変わらない）
```

## タグの組み合わせ例

### 色+フォントサイズ

```
<#ff0000><s80>大きな赤い文字<s><#>
```

### 段階的な変化

```
<s20>小<s40>中<s60>大<s>
<rot15>少し<rot30>もう少し<rot45>かなり<rot>戻る
<ls0>つめる<ls10>標準<ls30>ひろめ<ls>戻る
<@メイリオ>メイリオ<@HGPゴシックE,BI>HGゴシックE太斜<@>戻る
```

### 表示演出

```
<r5>ゆっくり表示...<w2>（2秒待機）<r0>ドン！<c2>次の場面へ
```

### 座標+色+回転の組み合わせ

```
<#ff0000><p+0,-50><rot-15>警告！<rot><p+0,+0><#>
```

### ルビ+装飾の組み合わせ

```
<#ff0000><s80><rb東京,とうきょう><s><#>は<rb日本,にほん>の首都です
```
→ 「東京」は赤色80ptでルビ付き、「日本」は通常色でルビ付き
